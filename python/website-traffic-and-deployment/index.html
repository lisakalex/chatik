Website Traffic Categories

  • Low traffic: ~100–5,000 monthly visitors (personal blogs, local businesses).

  • Moderate traffic: ~10,000–100,000 monthly visitors (niche blogs, small e-commerce).

  • High traffic: ~500,000–1M+ monthly visitors (popular websites).

  • Very high traffic: 50M+ monthly visitors (Google, YouTube, Facebook).

Django Scalability

  • Django can scale to very high traffic with proper deployment.

  • Key factors: Gunicorn, Nginx, caching, database optimization.

  • Performance ranges:

    • On a modest VPS: ~500k–2M visits/month.

    • With scaling: 50M+ visits/month.

  • Examples: Instagram, Disqus, Mozilla Addons.

Apache2 with mod_wsgi

  • Works with Django, but generally less efficient than Gunicorn + Nginx.

  • Modes:

    • Daemon mode: Runs in separate processes, offers better isolation, security, and tuning (recommended).

    • Embedded mode: Runs inside Apache, less secure and less scalable.

  • Check setup: Look for WSGIDaemonProcess and WSGIProcessGroup in Apache config.

Gunicorn + Nginx Deployment (Recommended)

  1. Install Gunicorn.

  2. Test locally.

  3. Create a systemd service for Gunicorn.

  4. Configure Nginx as reverse proxy (connect to Gunicorn socket).

  5. Collect static files.

  6. Secure with Let’s Encrypt SSL.

Traffic Load & Stress Testing Tools

  • ApacheBench (ab) – simple benchmarking.

  • wrk – modern, high-performance.

  • Locust – simulates user behavior.

  • k6, Siege – alternatives.

  • Monitoring: use htop, dstat, logs to track CPU, memory, errors.

Ethical Hacking (Kali Linux + MySQL)
Practice only in controlled environments: personal VMs, TryHackMe, HackTheBox.

Testing process:

  1. Set up a test MySQL server with weak credentials.

  2. Scan for ports/versions (nmap).

  3. Attempt brute-force (hydra).

  4. Connect with MySQL client.

  5. Test for SQL injection (sqlmap).

MySQL Hardening Tips:

  • Disable remote root login.

  • Enforce strong passwords.

  • Apply security patches.

  • Restrict access & follow least privilege.
